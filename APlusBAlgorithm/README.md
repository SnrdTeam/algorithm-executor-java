# A + B

Простой пример *Алгоритма* вычисления суммы двух входных значений: слагаемых.

## Входные данные

*Определение задачи* должно содержать единственный текстовый файл input.txt, в котором записаны в текстовом представлении два слагаемых, разделенные пробелом. Например,
 
    35 7
    
## Выходные данные

Решением задачи является строковое представление числа - суммы слагаемых, указанных во входных данных. Публикуемому решению устанавливается статус конечного (Final).
